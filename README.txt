Ispirithalaya is free and open source project for implement hospital system as easy way.

Following function are included in the ispirityalaya.

    Online Doctor Channelling
    Online Payments (with out gate way)
    Chatting
    Online booking for checkup
    Booking time for labtest & checking whether the reports were ready
    Room reservations

Requirement.

Apache 2.2.x
Php 5.3.x
Mysql 5.1.x
Zend Freamworks 1.11.x (Zend lib and ZendX lib)
